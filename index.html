台灣餐廳隨機推薦器

LocationIQ (Geocoding) + Nominatim (Fallback) • Overpass (餐廳資料)

程式使用說明

電腦版使用方式

1. 選擇國家 / 縣市 / 區域 / 街道 / 店家類型。

2. 街道 / 門牌可選填，填寫後「搜尋半徑」才會顯示，系統依據街道中心搜尋附近餐廳。

3. 點擊「搜尋餐廳」隨機抽選餐廳。

手機版使用方式

1. 可手動選擇國家 / 縣市 / 區域 / 街道 / 店家類型。

2. 填寫街道 / 門牌後，才會顯示「搜尋半徑」，系統將依據街道中心搜尋附近餐廳。

3. 或點擊「📍 取得我的位置」，系統會以你所在位置為中心搜尋附近餐廳，並顯示搜尋半徑供調整。

4. 點擊「搜尋餐廳」隨機抽選餐廳。

資料提醒

店家資料來源包含 OSM、LocationIQ、Nominatim，部分資料可能缺失或不完全正確，請自行查核。
